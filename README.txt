This plugin allows you to integrate the Magnet Audience Engagement Solution very easily on any WordPress powered website.

== Description ==
Using Magnet you can increase visitors' engagement, improve your website's SEO, and understand which topics/names/categories of posts your visitors are mostly interested in reading about.

You can specify the location or section you would want each of Magnet's features included on based on your website's theme.

The features that Magnet provides are:
1) Summary
2) Related Articles
3) Entities (list or in-text links)
4) Entity Related Articles Page
5) Follow Story
6) Follow Entity
7) Personalized Newsletter
8) Recommendations
9) Meta-Tags & Description
10) Smart Analytics

along with a Client Dashboard (http://connect.klangoo.com).

Magnet supports 53 languages as listed at: www.klangoo.com/SupportedLanguages.pdf.

For further information about the Magnet Audience Engagement Solution please visit http://www.klangoo.com.


== Installation ==
1. Upload \"WP-Klangoo\" using your WordPress CMS, or you can directly download it from the WordPress Plugin Directory by searching for \"Magnet\".
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Go to \"Magnet settings\" from your admin menu, and enter the settings as directed.
4. Depending on the used theme, you can place any \"Magnet\" widget inside the placeholders that the theme exposes.
